# Импорт и отображение логов по адресу получателя

1. Выполнить в MySQL создание таблиц в файлом create_tables.sql, при необходимости настроить подключение к БД в gzprmapp.pl в строке 11
(Схема немного изменена относительно исходной)

2. Запустить импорт логов import_log.pl

3. Запустить веб приложение Mojolicious::Lite так:
morbo gzprmapp.pl

Здесь http://127.0.0.1:3000 будет страница формы поиска логов по адресу получателя.
